# Local Cursor directory

A local reference for Cursor rules and resources used in this project. For more rules and MCP servers, see [cursor.directory](https://cursor.directory/).

---

## Project rules (in `.cursor/rules/`)

All rules live in a single file:

| File | Description |
|------|-------------|
| **project-rules.mdc** | Project rules and conventions. Edit this file to add or change rules (e.g. Quarto-only reports, style, patterns). |

---

## Adding more rules

1. Open `.cursor/rules/project-rules.mdc` and add a new section (e.g. `## Your rule name`).
2. Or browse [cursor.directory/rules](https://cursor.directory/rules) and paste rule content into a new section in `project-rules.mdc`.
