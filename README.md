# BP v3.0 TM

R + Quarto project: targets pipeline and analysis reports.

## Folder structure

```
├── analysis/           # Data, outputs, and pipeline
│   ├── data/           # Raw and processed data (git-ignored)
│   ├── output/         # Generated outputs
│   │   ├── models/     # Fitted models (e.g. .rds)
│   │   ├── plots/      # Figures (600 DPI)
│   │   └── tables/     # Tables
│   └── scripts/        # Source code and pipeline
│       ├── 1_targets.qmd   # Targets pipeline + report (Quarto)
│       ├── pipeline/       # Pipeline definition and cache
│       │   ├── _targets.R      # Generated by targets
│       │   ├── _targets/       # Pipeline cache (git-ignored)
│       │   └── _targets_r/     # Generated target scripts (git-ignored)
│       └── setup/
│           └── install_packages.R
├── docs/               # Project documentation
├── .cursor/            # Cursor rules
├── .cursorignore
├── .gitignore
└── Project.Rproj
```

## Quick start

1. **Install R dependencies**  
   `Rscript analysis/scripts/setup/install_packages.R`

2. **Run the pipeline**  
   `cd analysis/scripts/pipeline && Rscript -e "targets::tar_make(reporter = 'verbose')"`

3. **Render the report**  
   `quarto render analysis/scripts/1_targets.qmd`

4. **Open the report**  
   `analysis/scripts/1_targets_report.html`
